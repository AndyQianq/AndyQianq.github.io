    • 一月份b站的新番，目前已经更新到第八集，没有买b站的会员，所以我只看到了第七集。
    • 整体而言是一部不错的番，第一集很惊艳，从画质，构图，颜色，到配乐，声优都很棒。每一集都有亮点和泪点，从第一集到第七集也可以看到主人公的成长，变化，虽然每一集都可以当成一个小剧场来看，但是整体的连贯性也还不错，除了在第五话让人觉得有一种进步太大剧情跳跃的感觉以外，基本没有大问题。
    • 主要是对于作为一个颜党对我来说，这种程度的画面还是完全受用的，一口气看到了第七集，中间都没有中断，给人的感觉很舒适。但是如果是追番的话，可能我没有那么大的兴趣看完，对于我来说这种番还是比较适合一口气看完哒。
    • 整体属于治愈系，虽然是战后题材，但是画面和配乐给人的感觉都是很暖的。在剧情方面也是，没有什么故意搞笑的地方，但是也不至于很严肃。
    • 人设方面可能有一点老套，一开始看到女主的人设，总给我感觉很像圣女贞德。（原谅我还没有完完整整的看完Fate）还有印象比较深刻的是，女主的眼睛，忧伤清澈的感觉，加分加分！！然后就是战后的背景，总会让我把女主往剑心的人设上靠。
    • 女主从性格上来看不属于特别讨喜的，但是绝对是属于那种男女通吃型的。看番的时候，整个感觉有种在恋爱的感觉。（我是女的我是女的我是女的）
    • 总线剧情简单，预测不出意外的话，无非就几种可能，应该不会出太大的坑。前三集都是很正常的从女主身边其他人的视角，但是提供的信息只有——薇尔莉特和少佐之前的情愫。后面比较喜欢的剧集就是第六话的情节，尤其是后期薇尔莉特把头发束起来之后，那根红发带又激起“你的名字”情结。特别是六话中间的情节，给人一种恋爱的感觉尤为强烈。
    • 然后感觉比较好的就是第七话的，最后那一段薇尔莉特撑伞朝作家走过来的那一段简直美爆了，感觉到制作的诚意和经费在燃烧。
    • 看了上面那么多废话，终于扯到了主题——打字机。不知道是不是作了这么多年的工科女，对于机械之类的东西有一种情结。所以看番看到现在对我印象最为深刻的居然是打字机。

打字机的历史

  • 世界上的第一台打字机诞生于1808年，是由意大利人佩莱利尼·图里发明的。
  • 发明的理由是爱情。
  • 佩莱利尼·图里发明打字机的动机是为了自己的一位失明的女友。故事的女主角名字叫卡洛琳娜。博主在搜查资料的时候，看到了由美国作家华莱士写的《盲眼女伯爵的打字机》，对这段故事有兴趣的读者可以前去观看。
  • 但是佩莱里尼·图里发明的那一台打字机并没有保留下来，所以我们无法知道它的外形和结构了。
  • 首次投入商品化生产的英文打字机是由丹麦哥本哈根的尤尔根斯机械公司出品的，发明人则是一名叫做马林·汉森的神父。但是这台机器并没有投入生活化的使用。

  • 第一台真正实用的打字机的发明人是克里斯托弗·拉森·肖尔斯。在19世纪60年代，美国人肖尔斯制作了一架木制的打字机模型。

  • 所以打字机从真正意义上投入使用已经是1874年了，在1927年，开始成为平常之物了。

  • 此后一家武器制造商——雷明顿公司从肖尔斯那买下了打字机的生存权，并在1873年开始投产，3年就销售了数千台打字机。
  • 到了20世纪70年代后期，美国开始出现了电脑打字机，与传统的机械打字机不同，它不仅可以打字，而且还具有存储、修改、编辑的功能。此后，随着个人计算机的发展，电脑慢慢取代了机械打字机。但是庆幸的是我们还是可以从现如今的键盘上字母的分布看出机械打字机对留下来的影响。
  • 现在怀旧的人还是会选择在家里放置一台打字机，它就好像岁月留下的痕迹，像小时候妈妈那一台刻着花纹的缝纫机。敲打在打字机上的键盘上发出的“哒哒哒”声成为了怀念当时的一种情怀。
  • 最后在2012年11月21日。brother公司在北威尔士的打字机生产工厂宣告关闭，这就意味着最后一台下线的打字机将作为一个时代的标志物直接进入伦敦科学博物馆。自1985年以来，该公司已经生产了590万台的Wrexham打字机。经过130年，最终被电脑彻底替代。
  • 好了历史就写到这里了。然后这里整理了几个问题，我想大家应该都是比较想知道的。

  • 问题一：现在还有生产机械打字机的工厂吗？
    问题二：打字机上的字母排序为什么是QWER？

  • 问题一的解答：
    • 目前，大的生产商基本已经放弃生产了，但是在某宝上，或者ebay上还是有人贩卖二手的打字机。保养很好的打字机现在基本都是可以正常使用的。
    • 之前看到了日本还有生产一种便签迷你打字机，但是个人觉得没有什么用处，而且造型上也不是很复古。
  • 问题二的解答：
    • 原来，早期机械打字机都是依靠按键驱动后方的字锤，这种设计面临一个常见的问题：当打字速度太快时，前一个字母的字锤还未完全弹起，后面一个字母的字锤就已经落下，两个字锤很容易搅在一起，发生“卡壳”。所以，Scholes的键盘有意把频繁在单词中连续出现的字母（比如S和T）分散开来，并且把一些常用的字母安排给了力量很弱的小拇指和无名指（比如A和O），而最中间一行的字母使用率仅有30%左右，打字时手指被迫上下跳动。此外，在一个使用右手更为普遍的世界，Scholes却把主要的击键任务安排给了左手——在QWERTY键盘上，左手的劳动大约占了57%。
    • 通过这些反效率的设计，Scholes发明的打字机键盘更少出现卡壳现象。著名的军火制造商Remington家族看中了这一设计，买下了他的专利并花大力气推向了市场，奠定了QWERTY键盘被广泛接受的基础。
    • 读到这里，细心的读者可能会仔细观察自己的键盘，并且留意到一个奇怪的事实：既然频繁在单词中连续出现的字母被有意分开了，那为什么E和R会出现在一起？要知道，在英文中，这两个字母经常一起出现，作为形容词比较级（比如higher）和表示职业名称（比如teacher）的后缀。
    • 其实，在Scholes的设计中，R并不在最上面一行，它是被Remington公司挪上去的。原因？挪上去之后，”typewriter”（打字机）这个单词就可以用第一行的按键完整打出来了。这样，在推销产品时，就多了一个噱头。

打字机的品牌

  • corona（歌罗娜）

    • 由Brother公司出品，1902年款有一台折叠式打字机，小巧玲珑，在上世纪20年代，在上海流行，宋庆龄、陈天华等社会名流都使用过。
    • 1991年该公司还推出了simple smart的pc机，但是在海外的竞争激烈，最终拖垮了公司的财政，最有corona公司在1995年申请破产，走完了92年的光辉历程。
    • 在某乎上看见的，答主回复在ebay上购买，运费280rmb，机器900rmb
    • 型号为corona3

  • Olivetti（奥莉薇特）

    • 还有一个很接地气的名字——好利获得
    • 已经有100多年的历史了，成立于1908年10月29日，创始人Camillo Olivetti，起源于意大利。
    • 虽然看着十分接地气的样子，但是价格好像并不是很亲民。
    • 以上图片的打印机型号为Olivetti Lettera 32
    • 这一台的价格并不是很清楚，但是Olivetti Lettera 22的价格看到一些网上标价是3500一台。

  • underwood（安德伍德）

    • 从1874年安德伍德家族开始生产打字机色带和复写纸，在1896年到1900年出品了underwood1号和underwood2号。
    • 具体的图片参考上面。在紫罗兰的永恒花园中，女主使用的就是他们家的打印机。
    • 价格和Olivetti是差不多的，可能会便宜一点。
    • 样子还是非常复古好看的，就是感觉打字的效果没有那么好。

  • 飞腾

    • 国产品牌
    • 样子上是仿照Olivetti的机型，这也是为啥你看Olivetti的机械打字机那么接地气的感觉。这种在某宝上还是可以买到的，价格也不会很贵，保守的估计应该在1000块以内可以搞定。
    • 打字的效果主要看保养的程度了。

  • 英雄

    • 我大英雄牌果然闪亮闪亮最闪亮，哈哈哈哈哈
    • 原来除了钢笔，墨水，英雄同样还会生产打字机，貌似这个和飞腾一样已经成为一代人的回忆了。

  • 最后还是总结一下吧

    • 总的来说毕竟是被时代淘汰的东西了，不过主要看个人喜好吧，但是还是觉得花好几千块钱买一个旧的打不了字的废铁回来没有什么意义。
    • 当然了，如果您是纯粹从收藏角度来看的话，自然花多少钱都是看个人乐意了呗。

  • 最后的最后，再说一遍就瞅着好看的画面，治愈系的剧情，Violet Evergarden我追定了。
