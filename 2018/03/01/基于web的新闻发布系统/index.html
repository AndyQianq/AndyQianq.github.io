如何在mac上配置Apache Tomcat的详细步骤

第一步 下载Tomcat

下载网址

  • 选择Core：zip下载
  • 放置到/Users/XXX下（或者别的地方也可以，只要你能记住就可以）

第二步 启动Tomcat

  • 选择到终端启动Tomcat
  • 在终端输入安装路径：/Users/XXX/apache-tomcat-7.0.85/bin
  • 然后在bin下输入：./startup.sh+回车
  • 出现以下结果：

    Using CATALINA_BASE: /Users/XXX/apache-tomcat-7.0.85
    Using CATALINA_HOME: /Users/XXX/apache-tomcat-7.0.85
    Using CATALINA_TMPDIR: /Users/XXX/apache-tomcat-7.0.85/temp
    Using JRE_HOME: /Library/Java/JavaVirtualMachines/jdk1.8.0_144.jdk/Contents/Home
    Using CLASSPATH: /Users/XXX/apache-tomcat-7.0.85/bin/bootstrap.jar:/Users/XXX/apache-tomcat-7.0.85/bin/tomcat-juli.jar
    Tomcat started.

  • 如果你的终端那么恭喜你的Tomcat已经启动

  • 如果不幸，并没有以上的结果，那么请参考一下的情况对照找出解决方法

    1.出现了：类似 “Permission denied” 字样
    2.出现了：-bash: startup.sh: command not found

  • 第一种情况请看：

    • 重新打开终端
    • 输入sudo chmod 755 xxx/bin/*.sh +回车；
    • 再重新打开终端，重复以上的步骤，再测试是否成功
  • 第二种情况请看：
    • 确定自己输入的确实是“ ./startup.sh”
    • 如果不是直接重新输入一遍

第三步 监测Tomcat配置启动是否成功

  • 打开你的safari，然后输入：http://localhost:8080/(或者直接打开我此处提供的链接)
  • 如果出现tomcat首页，那么恭喜成功。
  • 如果没有，那么请重新回到开头查看哪里出现了问题。
