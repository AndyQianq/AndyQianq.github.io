正文部分

关于鲸落

主题

  • “你知道这世界上有 一种现象叫‘鲸落’吗？鲸鱼死后尸体沉 在海底下，供养其他水生物仍能生活几 十年，在我看来就像一个温柔的奇迹， 用自己的身体，建立起一个孤独的岛屿。”
    ——《深海中的温柔孤岛》

  • 全世界鲸目物种有80多个，但是只有达到30吨级别的大型鲸才能真正形成繁盛的鲸落——这就只剩下不到十种，而其中一半是濒危的。过去两百年里，工业化捕鲸将大型鲸推入了十分危急的境地，今天全球海洋里的鲸落数量，可能只有以前的不足六分之一。

准备工作

新建文件

  • 我是选择A4纸格式、分辨率定为300

  • 这里说一下，如果要考虑图片的打印效果，分辨率默认是300，但是如果是平时作图，不用打印的话，72的分辨率就绰绰有余了

关于构图

  • 既然是鲸落，首先想到的就是鲸鱼啊!
  • 在资料的部分我讲过，座头鲸比较适合作画。

  • 原因如下：

    • 1.体型大但是结构匀称，不想抹香鲸显得头重脚轻。
    • 2.色彩统一，不像虎鲸的颜色对比太过于显眼。
    • 3.座头鲸又叫做“大翅鲸”，鱼鳍宽大，很容易让画面显得飘逸。

  • 如果画面只有一条鲸鱼，一来画面单调，二来没有参考物，读者感受不到鲸鱼的庞大.

  • 就像学《逍遥游》的时候

    • 北冥有鱼，其名为鲲,鲲之大不知其几千里也。

  • 文字可以用几千里来做比较，但是画面只有A4纸的大小。

  • 于是，我加了一个少女的形象来作为参照。
  • 建议画图的时候，打开导航器，就可以直观的感受画面效果。

    主要工作

    环境色

  • 粗略的完成线稿之后，开始铺色，对画面进行定调,可以使用大面积的喷枪上色

  • PS:但是对于ps来说，当笔的像素跳到1000以上之后，会有很可怕的延迟然后电脑开始嗡嗡作响，有些老式机会直接停机，尤其是联想的。。。真的，再好的pc机都会出现这种现象，不要问我怎么知道的

  • 这里稍微提一下：
    在笔刷这一部分真的还是专业的绘图软件比较好，毕竟ps只是图像处理工具

  • 推荐的绘图工具有：

    • sai
    • painter

  • 但是以上的软件也有自己的缺点

    • 例如sai在厚涂方面表现一般，笔触过于平滑，很难处理细节，色彩上也过于小清新，表现的风格会收到局限。
    • painter则是专业的绘图软件，但是占用的内存较大，颜色丰富，笔触细腻，问题是对于刚刚入门的萌新们，看到那么多的功能会很懵逼的好嘛（躲在墙角瑟瑟发抖）。
    • 另一方面，painter在大陆的受众比较少，在网上找资源教程什么的比较难，台湾等部分地区使用比较广泛。

  • 如果不用大笔刷铺色的话，其实ps里还是用其他很好用的工具可以代替的

  • 这里我使用了工具栏处的“渐变”处理,对角线一拉，马上自然过渡

  • 然后在用笔刷增加一点笔触进去

光效果

  • 例如加入一下水波的痕迹，然后开始加入发光的效果（效果如下）

  • 为什么这么早加入发光特效

    • 因为这样对于后面展现主题的细节有好处啊

    • 有了光之后，你就能知道在鲸鱼的什么地方是暗面，什么地方是亮面了啊。。。这样该有高光的地方你就加去嘛

    • 对于小白，这一招还是蛮好用的

  • 好了，说一下发光特效怎么加

    • 首先选一张深海的光源图

    • 例如

  • 然后选择你想要的光源部分

    • control+c
    • control+v

  • 复制得到的图层命名为“光源”

  • 放在线稿的下面，铺色的上面

  • 然后在图层效果里选择——颜色加深

细节上色

  • 首先完善一下鲸鱼的感觉，塑造一下明暗面：

  • 这里新建一个图层，用模糊笔刷浓度调节到40～50%之间，给鲸鱼的脊背上色。

  • 高光的部分用30号笔刷上色。

  • 通过光线来判断亮面和高光的位置。
  • 现在我们再处理一下人物，虽然这幅图的人物不是重点。
  • 我们来看一下这个略low的小人。

  • 现在我们给小人上一些颜色。
  • 注意上色的时候不能用过于鲜艳的颜色，不然会显得画面很突兀。

  • 现在我们看看这幅图的整体效果：

  • 审视一下有没有什么问题，然后重头戏就来了——为画面加入细节。

  • 一幅画一眼看上去颜色不能辣眼睛，这是刚刚一些列做的事

  • 而一幅画到底耐不耐，这里继续在画面的主体（鲸鱼）上加入细节，这里就可以好好利用鲸落的主题了，就是那种虽死犹生的衰败的美感嘛（原谅我糟糠的语言表达。。。）

  • 加入细节的地方要用更细腻的笔触，这里选用的是25号笔刷加入纹路。

  • 一点一点的擦去线稿里粗糙的线条,用30号笔刷制造伤痕。

  • 其实细节还可以再加很多，例如后期看可以为了让害民啊看起啦更真实，在加入气泡，血丝，小鱼之类的,这里由于时间的关系就不多说了。

  • 如果只是单纯的一幅图的画，（好吧作为一个完成度不是特别高的图，例如小人那个地方我就比较偷懒。。。）也就差不多了。

关于字体

  • 首先在素材包里打开字体图片

  • 然后在右下角处面板点开通道页

  • 点击右下角的将通道作为选区载入

  • 再点击菜单栏的选择——》反选

  • 接下来就是复制、粘贴了

  • 粘贴到新建的文件后会在原先的背景上新增图层，也就是我们刚刚复制的文字图层。
  • 但是文字太小了，所以这里进行放大文字。

  • 点击菜单栏——》编辑——》变化——》缩放
